# Research: DNS Server Market Landscape
**Date:** 2026-06-08
**Status:** Finalized

### Objective
Analyze popular open-source DNS server solutions for home/self-hosted environments.

### Market/Context Analysis
The market is dominated by tools focusing on ad-blocking, privacy, and ease of use.
- **Pi-hole:** Community favorite, extremely stable, lightweight. Excellent for home-wide ad-blocking.
- **AdGuard Home:** Modern alternative with a superior web dashboard and native support for encrypted DNS protocols.

### Key Alternatives
- **Technitium DNS Server:** Powerful, cross-platform, handles recursive and authoritative duties.
- **Unbound:** Lightweight caching resolver, excellent performance.
- **SmartDNS:** Optimized for query response speed.

### Conclusion & Next Steps
For our project's goal of self-hosted DNS, leveraging the features of AdGuard Home (dashboard/UI focus) and Pi-hole (stability/analytics focus) serves as a benchmark for user expectations.
Next steps: Define our specific differentiator within this landscape.
